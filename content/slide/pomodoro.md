# 工作時間
- 平日每天約4小時
- 假日每天約6小時
- 合共：
  `4hr x 5day + 6hr x 2day = 32 hrs`
- 如何保持專注?

# 規劃
- 每科應用15分鐘回顧及整理當天所學
- 功課應盡早完成
- 每天用盡所有工作時間
